# Simple base framework



### Get Simple:



0. Just clone repo:

  ```sh
$ git clone https://github.com/artekha/Simple simple-base
  ```

1. Enter to folder:

  ```sh
  $ cd simple-base
  ```

2. Install npm;

  ```sh
  $ npm install
  ```

3. Install gulp;

  ```sh
  $ gulp install
  ```

#### Good luck!
